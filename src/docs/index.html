API Information

  • Base Url : https://rest-api-mongoexpress.vercel.app/
  • Limit Request : 1000 / 1 day ( cek ga jebol jir )

Endpoints

Method Endpoint Description
GET /api/v1/products/ Get all products
GET /api/v1/products/show/{slug} Get a product by slug
POST /api/v1/products/ Create a new product
PUT /api/v1/products/update/{slug} Update a product by slug
DELETE /api/v1/products/delete/{slug} Delete a products by slug
Example using javascript to get all products
                        
fetch('https://rest-api-mongoexpress.vercel.app/api/v1/products/')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error(error))
Response
                        
{
    "status": 200,
    "message": "Products retrieved successfully",
    "data": []
}
Example using javascript to get a product by slug
                        
fetch('https://rest-api-mongoexpress.vercel.app/api/v1/products/show/example-name-product')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error(error))
Response
                        
{
    "status": 200,
    "message": "Product retrieved successfully",
    "data": []
}
